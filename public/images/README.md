# Images folder

Drop all site images into this folder. Keep originals small and web-optimized.

## Recommended image sizes

| Purpose | File | Suggested size | Format | Max file size |
| --- | --- | --- | --- | --- |
| Homepage hero headshot | `teri-headshot.jpg` | 1200 × 1400 px | JPG or WebP | ~300 KB |
| About page portrait | `teri-about.jpg` | 900 × 1100 px | JPG | ~250 KB |
| Open Graph / social share | `og-image.jpg` | 1200 × 630 px | JPG | ~200 KB |
| Post header images | — | 1600 × 900 px | JPG | ~250 KB |
| Endorsement thumbnails | — | 200 × 200 px | JPG | ~40 KB |
| Community photos (gallery) | — | 1600 × 1100 px | JPG | ~300 KB |

## Tips for non-technical editors

- Use [squoosh.app](https://squoosh.app/) (free, browser-based) to compress images before uploading.
- Always add **alt text** when adding images via `/admin` — it describes the image for people using screen readers and helps SEO.
- Keep filenames lowercase with hyphens (e.g. `teri-at-terry-fox-run.jpg`), no spaces.

## Image types specifically needed (placeholders to replace)

1. **Homepage headshot** — friendly, warm, outdoor if possible
2. **About page portrait** — candid, at a community event or with family
3. **Hero/banner alternate** — horizontal community shot of Teri with neighbours
4. **3–5 community photos** — Teri volunteering, at events, walking trails
5. **Endorser headshots** — small square crops of each person endorsing
6. **Open Graph image** — the image that shows when the site is shared on Facebook/Twitter (can reuse headshot)

Placeholders are in `/public/images/placeholders/`.
